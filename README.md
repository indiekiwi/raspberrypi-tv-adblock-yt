# Raspberry Pi Ad Mute & Skip Automation for YT

This project automatically mutes the TV and skips YT ads using a Raspberry Pi. The idea is to use IR communication and optical character recognition (OCR) for ad detection and control.

## Features:
- Mutes the TV when an ad plays.
- Clicks the "Skip" button when it becomes available.

## Components:
- Raspberry Pi SBC
- IR Receiver
- IR Transmitter
- Webcam
- Raspberry Pi Pico (Used during IR proof of concept testing)
